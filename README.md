## Project Overview

**Addis Photo & Media Agency** is a full-stack web application designed to provide a modern and professional online presence for a photography and media company. The platform showcases the agency’s creative services, portfolio, and contact information, making it easier for clients to explore work samples and connect with the business.

This project highlights a clean, responsive, and visually appealing digital experience that helps improve service visibility, portfolio presentation, and client communication through a well-structured full-stack web platform.

---

## Features

- **Professional landing page with agency introduction**
- **Photography and media services showcase**
- **Portfolio / gallery section for visual projects**
- **Contact section for client inquiries**
- **Clean and user-friendly navigation**
- **Responsive design for desktop, tablet, and mobile**
- **Fast and interactive frontend experience**
- **Backend integration for handling project structure and functionality**
- **Well-organized full-stack project structure**

## Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/Haimanot515/Addis-Photo-an-Media-Agency.git
```

### Navigate to the Project Folder

```bash
cd Addis-Photo-an-Media-Agency
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

Open a new terminal and run:

```bash
cd backend
npm install
npm start
```
